# ansible_learning

This is my ansible project that is intended to give me experience and practice using ansible. 


## Project Setup

### Make sure you are in the ansible_learning directory
```
cd ansible_learning
```

### Create the venv 
```
python3 -m venv venv
```

### Activate the Environment
```
source venv/bin/activate
```

### Install Ansible
```
pip install ansible
```
